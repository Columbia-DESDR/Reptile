# FIST

## Dependencies
Flask
Pandas

## Unix Bash (Linux, Mac, etc.):

```
$ export FLASK_ENV=development
$ export FLASK_APP=app.py
$ flask run
```

## Windows CMD:

```
> set FLASK_ENV=development
> set FLASK_APP=app.py
> flask run
```

## Windows PowerShell:

```
> $env:FLASK_ENV = "development"
> $env:FLASK_APP = "app.py"
> flask run
```
