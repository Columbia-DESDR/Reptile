## Getting Started

### Installing

* Install dependencies:
    * `pip install -r requirements.txt`
* Set up flask environment::
    * Windows: `set FLASK_APP=app.py`, `set FLASK_ENV=development`
    * Mac/Linux: `export FLASK_APP=app.py`, `export FLASK_ENV=development`
* Start the site:
    * `flask run`
