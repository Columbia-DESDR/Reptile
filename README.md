# FIST

## Dependencies
python3.6

Flask

Pandas

## Unix Bash (Linux, Mac, etc.):

```
$ export FLASK_ENV=development
$ export FLASK_APP=app.py
$ flask run --host=0.0.0.0
```

## Windows CMD:

```
> set FLASK_ENV=development
> set FLASK_APP=app.py
> flask run --host=0.0.0.0
```

## Windows PowerShell:

```
> $env:FLASK_ENV = "development"
> $env:FLASK_APP = "app.py"
> flask run --host=0.0.0.0
```
